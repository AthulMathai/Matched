# Matched

Matched is an 18+ global video chat MVP that connects people through country, language, gender preferences, and shared interest tags.

## Core Features

- Supabase authentication
- 18+ onboarding flow
- Profile preferences
- All-country dropdown
- Full language dropdown
- Default tags plus custom typed tags
- 30-second pre-match ad gate
- Matching queue
- Video chat room placeholder
- Camera/microphone controls
- Face-clear camera guard
- Skip/report/block
- Chat history only for 10+ minute sessions
- Ping previous users from history
- Messaging after ping acceptance
- Blocked users page
- Admin dashboard
- Report review
- Session audit

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Supabase Auth
- Supabase PostgreSQL
- Supabase Realtime-ready messaging
